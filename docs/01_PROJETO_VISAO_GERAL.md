## 4. ESQUEMA FIXO DE ANÁLISE POR SALMO

Cada Salmo segue obrigatoriamente esta estrutura trifásica:

### PARTE I — FUNDAMENTOS
- **4.1** Análise letra por letra (cada letra hebraica mapeada a uma Sefiráh e função)
- **4.2** Tzerufim (permutações de letras revelando significados ocultos)
- **4.3** Guematria com decomposição Sefirótica
- **4.4** Análises adicionais de versículos-chave

### PARTE II — PRINCÍPIO OPERACIONAL
- Versículos mapeados com:
  - **Netiv** (caminho Sefirótico)
  - **Função**
  - **Sod** (segredo místico)

### PARTE III — ESTRUTURA GLOBAL
- Ciclos Temáticos
- Estrutura Global
- Sod Final (segredo místico supremo)

### SEÇÕES EDITORIAIS FINAIS (por Salmo)
- **APLICAÇÃO** — resumo prático extraído do conteúdo
- **KAVANÁH** — intenção espiritual sintetizada do texto
- **AUTOANÁLISE** — 1–2 perguntas derivadas do conteúdo
